# Step2: イベント候補抽出

1. `trend_keywords` を受け取る。
2. ドラマ化できそうなイベントを1〜3件選ぶ。
3. 各候補を「イベント名：一言説明」の形式で人間に提示する。
4. 人間の選択・修正を待つ。

## 制約

- 政治・宗教・差別・企業批判を回避する。
- 専門知識が必要な題材を避ける。
- ストーリー・感情・展開には一切言及しない（それは Step3 以降の仕事）。

## 介入点

- 使用するイベントの選択・修正を受け付ける。
- 確定後、次ステップ（step3-generate-situation）へ渡す。

## 出力

- `event_name`: 確定済みイベント名
