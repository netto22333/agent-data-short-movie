# **三本柱**

  

## **① 動画（YouTube / TikTok）**

  

**目的**

  

広告収入

  

**コンテンツの性質**

  

エンタメ寄り

  

**収益構造**

```
動画
↓
視聴
↓
広告表示
↓
広告収入
```

**必要能力**

- 注目を集める能力
    
- コンテンツ制作
    
- アルゴリズム理解
    

---

## **② AIOブログ**

  

**目的**

  

AI検索からの流入

  

**コンテンツの性質**

  

情報整理・解説

  

**収益構造**

```
検索
↓
ブログ記事
↓
情報提供
```

**必要能力**

- 情報整理
    
- 検索理解
    
- 記事構造
    

---

## **③ アフィリエイト**

  

**目的**

  

商品紹介による収益

  

**コンテンツの性質**

  

購買誘導

  

**収益構造**

```
商品紹介
↓
ユーザー購入
↓
成果報酬
```

**必要能力**

- セールス
    
- 商品理解
    
- 信頼構築
    

---

# **三本柱の能力構造**

|**柱**|**マーケ能力**|
|---|---|
|動画|Attention（注意）|
|AIO|Search（検索）|
|アフィリエイト|Persuasion（購買誘導）|

---

# **共通している構造**

  

すべて

```
コンテンツ
↓
トラフィック
↓
マネタイズ
```
